Question
Evaluate: $\displaystyle\int x^2 e^x\,dx$
1
Use Integration by Parts (ILATE): $u=x^2$ (Algebraic), $dv=e^x dx$ (Exponential)
$\int u\,dv = uv - \int v\,du$
2
First application: $u=x^2,\,v=e^x,\,du=2x\,dx$
$= x^2e^x - \int e^x\cdot 2x\,dx = x^2e^x - 2\int xe^x\,dx$
3
Apply by parts again to $\int xe^x\,dx$: $u=x,\,v=e^x$
$\int xe^x\,dx = xe^x - e^x$
4
Substitute back:
$= x^2e^x - 2(xe^x - e^x) = x^2e^x - 2xe^x + 2e^x$
âœ… Final Answer
$\boxed{e^x(x^2 - 2x + 2) + C}$
ðŸ”´ CBSE Tip: Always write +C at the end of indefinite integrals. Apply ILATE â€” Algebraic before Exponential. Show every step in By Parts.

Example 1
5 MarksBayes' Theorem
Question
A bag I contains 4 red and 4 black balls. Bag II contains 2 red and 6 black balls. One bag is chosen at random and a ball is drawn. Given the ball is red, find the probability it came from Bag I.
1
Define events: $E_1$ = Bag I selected, $E_2$ = Bag II selected, $A$ = Red ball drawn.
$P(E_1) = \frac{1}{2},\quad P(E_2) = \frac{1}{2}$
2
Conditional probabilities:
$P(A|E_1)=\frac{4}{8}=\frac{1}{2},\qquad P(A|E_2)=\frac{2}{8}=\frac{1}{4}$
3
Total probability P(A):
$P(A)=P(E_1)\cdot P(A|E_1)+P(E_2)\cdot P(A|E_2)=\frac{1}{2}\cdot\frac{1}{2}+\frac{1}{2}\cdot\frac{1}{4}=\frac{1}{4}+\frac{1}{8}=\frac{3}{8}$
4
Apply Bayes' Theorem:
$P(E_1|A)=\frac{P(E_1)\cdot P(A|E_1)}{P(A)}=\frac{\frac{1}{2}\cdot\frac{1}{2}}{\frac{3}{8}}=\frac{\frac{1}{4}}{\frac{3}{8}}=\frac{1}{4}\times\frac{8}{3}=\frac{2}{3}$
âœ… Final Answer
$\boxed{P(E_1|A) = \dfrac{2}{3}}$
ðŸ”´ CBSE Tip: ALWAYS: (1) Define all events clearly, (2) Write P(Eáµ¢) values, (3) Write P(A|Eáµ¢) values, (4) Compute P(A) using Total Probability, THEN apply Bayes'. Show all 4 steps for full marks!

âœ… Final Answer
Dimensions: $\dfrac{2r}{\sqrt{2}} \times \dfrac{2r}{\sqrt{2}}$ = $r\sqrt{2} \times r\sqrt{2}$ (a square!)

$\boxed{A = 8\sqrt{3} \text{ sq. units}}$

Ch 1: Relations & Functions ~5 Marks in Board
Example 1
5 MarksEquivalence Relation
Question
Show that R = {(a,b): 2 divides (a-b)} on Z is an equivalence relation.
1
Reflexive: a-a=0=20, so 2|(a-a). Hence (a,a)R for all a. R is Reflexive.
2
Symmetric: Let (a,b)R a-b=2k. Then b-a=-2k=2(-k) (b,a)R. R is Symmetric.
3
Transitive: (a,b)R and (b,c)R a-b=2k and b-c=2k. So a-c=2(k+k) (a,c)R. R is Transitive.
Conclusion
R is Reflexive + Symmetric + Transitive R is an Equivalence Relation.
CBSE Tip: Write all 3 properties separately. Each = 1 mark. End with "Hence R is an Equivalence Relation".

2 MarksPrincipal Value
Question
Find the principal value of $\cos^{-1}\!\left(-\dfrac{\sqrt{3}}{2}\right)$
1
Use: $\cos^{-1}(-x) = \pi - \cos^{-1}(x)$
$\cos^{-1}\!\left(-\frac{\sqrt{3}}{2}\right) = \pi - \cos^{-1}\!\left(\frac{\sqrt{3}}{2}\right)$
2
Since $\cos\frac{\pi}{6} = \frac{\sqrt{3}}{2}$:
$= \pi - \frac{\pi}{6} = \frac{5\pi}{6}$
Final Answer
$\boxed{\dfrac{5\pi}{6}}$
CBSE Tip: cos(-x) = π - cos(x). NOT negative! This is the most common mistake in ITF. Memorize this identity.

3 MarksSym + Skew Decomposition
Question
Express $A = \begin{pmatrix}3&5\\1&-1\end{pmatrix}$ as sum of a Symmetric and Skew-Symmetric matrix.
1
$A^T = \begin{pmatrix}3&1\\5&-1\end{pmatrix}$
2
Symmetric part $P = \frac{A+A^T}{2} = \frac{1}{2}\begin{pmatrix}6&6\\6&-2\end{pmatrix} = \begin{pmatrix}3&3\\3&-1\end{pmatrix}$. Check: $P=P^T$
3
Skew-Symmetric part $Q = \frac{A-A^T}{2} = \frac{1}{2}\begin{pmatrix}0&4\\-4&0\end{pmatrix} = \begin{pmatrix}0&2\\-2&0\end{pmatrix}$. Check: $Q^T=-Q$
Final Answer
$A = \begin{pmatrix}3&3\\3&-1\end{pmatrix} + \begin{pmatrix}0&2\\-2&0\end{pmatrix}$
CBSE Tip: Always verify P=P^T and Q=-Q. Diagonal of skew-symmetric is always zero!

5 MarksLinear DE
Question
Solve: $\dfrac{dy}{dx} + \dfrac{y}{x} = x^2$
1
Form: $\frac{dy}{dx}+Py=Q$ where $P=\frac{1}{x}$, $Q=x^2$.
2
I.F. $= e^{\int\frac{1}{x}dx} = e^{\ln x} = x$
3
Multiply both sides by I.F.:
$\frac{d}{dx}(xy) = x^3$
4
Integrate both sides:
$xy = \frac{x^4}{4} + C$
General Solution
$\boxed{xy = \dfrac{x^4}{4} + C}$
CBSE Tip: Steps: (1) Identify P, Q (2) Find I.F. (3) Multiply (4) Integrate. Always write +C. This exact format is what CBSE expects!

Question
If $\vec{a} = \hat{i}+2\hat{j}-3\hat{k}$ and $\vec{b} = 3\hat{i}-\hat{j}+2\hat{k}$, find $\vec{a}\cdot\vec{b}$ and angle between them.
1
Dot product:
$\vec{a}\cdot\vec{b} = (1)(3)+(2)(-1)+(-3)(2) = 3-2-6 = -5$
2
Magnitudes: $|\vec{a}|=\sqrt{1+4+9}=\sqrt{14}$, $|\vec{b}|=\sqrt{9+1+4}=\sqrt{14}$
3
Angle:
$\cos\theta = \frac{\vec{a}\cdot\vec{b}}{|\vec{a}||\vec{b}|} = \frac{-5}{14}$, so $\theta = \cos^{-1}\!\left(\frac{-5}{14}\right)$
Final Answer
$\vec{a}\cdot\vec{b} = -5$, $~\theta = \cos^{-1}\!\left(\dfrac{-5}{14}\right)$
CBSE Tip: Perpendicular dot product = 0. Parallel cross product = 0. Always find magnitudes separately before computing angle.
